Управление кошельками

Все созданные вами кошельки отображаются в панели под вашим именем.

Чтобы изменить активный кошелёк, достаточно просто нажать на кнопку с соответсвующим именем и приложение автоматически подгрузит связанные с данным кошельком данные.

Добавить новый кошелёк так же просто, как выбрать один из существующих. Просто кликните на символ add (Добавить кошёлек) на панели, расположенной слева от панели кошельков. В появившемся модальном окне введите желаемое название для кошелька и нажмите подтвердить. Далее созданный кошелёк появится на панели рядом с уже имеющимися и станет активным.

Для того, чтобы удалить имеющийся кошелёк, необходимо нажать на символ delete (Удалить кошелёк) рядом с кнопкой добавления кошелька. После этого, цвет созданных вами кошельков измениться на красный и после имени каждого из них появится небольшой символ clear, нажав на который вы сможете удалить данный кошелёк.

Помимо созданных вами кошельков, в панели кошельков вы также найдете кнопку Основной, при нажатии на которую приложение отобразит данные из всех кошельков, закрепленных за авторизованным пользователем. Однако, вы не сможете добавить новый элемент бюджета или вложение если эта кнопка активна. Для этого вам сначала придется вернуться в один из кошельков или создать новый.
